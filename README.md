

## Disney+ Clone

This is an react based application which is a clone of a popular OTT platform, Disney+Hotstar.
The front-end of application is react, for the backend firebase is used. The application uses google authencation for login.

## Installation
after navigating to the project directory first install all the node modules:

```bash
  npm install 
  cd my-project
```
    
## Starting the app

All commands to start and debug the app are outlined in the package.json file. To simply get it running run the following command while in your project repository.
```bash
     npm start
```
